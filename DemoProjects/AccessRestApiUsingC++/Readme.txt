// This c++ program is to get data from REST api 

1. Start visual studio
2. Create simple c++ console application program. It will have by default "Hello World"
3. Right click on project and goto "Manage nuget packages"
4. Search "cpprestsdk "  . This is library which provides methods to access webapi 
5. Install this package. 
6. Now copy contents of main.cpp in your project.
7. Start Web server providing REST api (Please run application from C# projects)
CreatingRestApi

8. Run this application
